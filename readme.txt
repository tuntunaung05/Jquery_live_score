1. Run the index.php file on the web brower (eg, http://localhost/bbtv/index.php).
2. Click on the name to select user to give 5 points.
3. Click on the "Give 5 points" button to increase the point of the selected user by 5. 

- leaderboard.json stores the scores.
- give_5_points.php is used for Ajax call which execute when user click "Give 5 points" button. 
- data.php is used to called by Jquery to synchronize the scores for all people who have that web page open.


Smarty template is use to separate HTML and PHP apart. 
All the template file (HTML code) can be found under templates folder.
